# 🤖 AI Search Agent

> An intelligent AI-powered search assistant built with **n8n**,
> **OpenAI Chat Model**, **SerpAPI**, and **Conversation Memory**.

## 📌 Project Overview

The AI Search Agent combines a Large Language Model with live Google
Search to answer user questions using the latest available information.
Unlike traditional chatbots, this agent can retrieve real-time
information and maintain conversational context.

## 🚀 Features

- Real-time Google Search
- OpenAI Chat Model
- Conversation Memory
- AI Agent Orchestration
- Workflow Automation with n8n

## 🛠 Tech Stack

## 📂 Workflow

```text
User Message
    ↓
Chat Trigger
    ↓
AI Agent
 ├── OpenAI Chat Model
 ├── Memory
 └── SerpAPI
        ↓
Generate Response
        ↓
Return Answer
```
